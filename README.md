I build real-time rendering systems from scratch in Vulkan and C++.

My final year project is a **Vulkan 1.3 renderer** built entirely around Dynamic Rendering,
explicit GPU synchronisation via `synchronization2`, and VMA-managed memory.
No `VkRenderPass`. No `VkFramebuffer`. Ever.

The stretch goal, 3D Gaussian Splatting, bridges directly into my PhD research on
neural radiance field representations and inverse rendering.

## Research Interests

My PhD (starting October 2026, supervised by Dr Conor Fahy) focuses on
**relightable neural representations from casual capture**. Core areas:

- Neural inverse rendering and material decomposition
- 3D Gaussian Splatting, NeRF, and hybrid representations
- Evaluation methodology for disentangled appearance
- Physically based shading and real-time light transport
- GPU-accelerated differentiable rendering
