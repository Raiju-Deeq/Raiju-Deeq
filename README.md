I build real-time rendering systems from the ground up. My final year project is a
**Vulkan 1.3 renderer in C++20**, built entirely around Dynamic Rendering, explicit
GPU synchronisation via `synchronization2`, and VMA-managed memory.
No `VkRenderPass`. No `VkFramebuffer`. Ever.

The Gaussian Splatting milestone bridges directly into my PhD research on neural
rendering and radiance field representations.
